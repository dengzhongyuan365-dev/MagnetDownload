# MagnetDownload GUI 设计规范

## 🎯 设计目标

创建一个现代化、直观的磁力链接下载工具GUI界面，参考qBittorrent、迅雷、Transmission等主流工具的最佳实践。

## 🎨 整体设计风格

### 设计原则
- **简洁明了** - 界面清爽，信息层次分明
- **功能完整** - 涵盖所有核心下载管理功能
- **响应式设计** - 支持窗口缩放和不同分辨率
- **现代化外观** - 使用现代UI设计语言

### 色彩方案
- **主色调**: 深蓝色 (#2C3E50)
- **辅助色**: 浅蓝色 (#3498DB)
- **成功色**: 绿色 (#27AE60)
- **警告色**: 橙色 (#F39C12)
- **错误色**: 红色 (#E74C3C)
- **背景色**: 浅灰色 (#ECF0F1)

## 🛠️ 技术实现方案

### GUI框架选择

#### 推荐方案: Qt6
```cpp
// 主要优势
- 跨平台支持 (Windows/Linux/macOS)
- 丰富的UI组件库
- 优秀的国际化支持
- 强大的信号槽机制
- 现代化的QML支持
```

#### 备选方案对比
| 框架 | 优势 | 劣势 | 适用场景 |
|------|------|------|----------|
| Qt6 | 功能完整、跨平台 | 体积较大 | 桌面应用首选 |
| GTK4 | Linux原生、轻量 | 跨平台支持有限 | Linux专用版本 |
| Dear ImGui | 高性能、游戏风格 | 学习成本高 | 性能要求极高 |
| Electron | Web技术、易开发 | 内存占用大 | 快速原型开发 |

### 界面响应式设计

#### 窗口大小适配
```cpp
class MainWindow : public QMainWindow {
private:
    void setupResponsiveLayout() {
        // 最小窗口大小
        setMinimumSize(800, 600);
        
        // 响应式布局
        connect(this, &QMainWindow::resized, [this]() {
            adjustLayoutForSize(size());
        });
    }
    
    void adjustLayoutForSize(const QSize& size) {
        if (size.width() < 1000) {
            // 紧凑模式：隐藏部分列
            downloadListWidget_->hideColumn(Column::RemainingTime);
        } else {
            // 正常模式：显示所有列
            downloadListWidget_->showColumn(Column::RemainingTime);
        }
    }
};
```

## 📱 用户交互流程

```
用户输入磁力链接
    ↓
点击"解析"按钮
    ↓
显示解析进度对话框
    ↓
解析完成，显示文件选择对话框
    ↓
用户选择文件和设置选项
    ↓
点击"开始下载"
    ↓
任务添加到下载列表
    ↓
开始下载并显示实时进度
```

### 2. 任务管理流程
```
右键点击下载任务
    ↓
显示上下文菜单
├─ 暂停/恢复
├─ 删除任务
├─ 打开文件夹
├─ 复制磁力链接
├─ 设置优先级
└─ 查看详细信息
    ↓
执行相应操作
    ↓
更新界面状态
```

```
点击"设置"按钮
    ↓
打开设置对话框
    ↓
切换不同设置页面
    ↓
修改配置选项
    ↓
点击"应用"或"确定"
    ↓
保存配置到文件
    ↓
应用新设置到程序
```
